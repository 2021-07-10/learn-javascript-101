## 3.4 数据类型

### 3.4.2 Undefined 类型

* 只有一个值 `undefined`
* 对于尚未声明过的变量，只能执行一项操作，即使用 `typeof` 操作符检测其数据类型
* 对于未初始化和未声明的变量执行 `typeof` 操作符都返回  `undefined`


### 3.4.5 Number 类型

* 使用 `IEEE754` 格式表示
* 八进制字面量在严格模式下是无效的
